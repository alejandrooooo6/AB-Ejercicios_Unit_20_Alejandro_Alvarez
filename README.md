# Unit 20

Base repository for the final AB project

Project build using Vite and Typescript, might add a website in the future.

### How to execute:

1. **Open a new terminal** inside `DataReader\src`.
2. **Execute the following command** through the terminal's console:
   `npx tsx 02-abstract-data-reader.ts --input-file estaciones_servicio_españa.csv`
3. **Once finished**, the following message will pop up in the terminal:
   `"Data correctly extracted and saved at DataReader\src\mis_gasolineras_filtradas.csv"`
4. **Navigate to that directory** to find and open the `mis_gasolineras_filtradas.csv` file containing all of the data.
